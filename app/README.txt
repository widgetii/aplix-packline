========================================================================

                            PackLine

========================================================================

1. ОПИСАНИЕ ПРОГРАММЫ

PackLine - программа управления упаковочной линией, предназначенная для
работы на терминалах обработки почтовых отправлений. В функции программы 
входят: приемка и контроль поступлений на вход упаковочной линии, слежение
за правильностью заполнения входных данных, помощь в упаковке почтового 
отправления, маркировка отправлений специальными кодами отслеживания, печать
сопроводительных документов. Программа может работать как на одной станции, 
выполняющей весь комплекс процедур, так и на разных станциях, отвечающих 
за конкретную процедуру приемки, упаковки или маркировки. Режим работы 
задается в конфигурации программы.

2. ИСПОЛНЯЕМАЯ СРЕДА

Приложение работает на платформе Java SE 7 с поддержкой Java FX 2.2.
Программа кроссплатформенная и может работать на разных операционных 
системах: Windows, Linux, Mac OS.

3. НАСТРОЙКА

Настройка программы сводится к редактированию конфигурационного файла 
/conf/packline.xconf. Это текстовый файл, имеющий структуру XML-документа. 
Обратите внимание, что файл записан в кодировке UTF-8. Для правильного 
отображения всех символов, используйте текстовый редактор с поддержкой UTF-8
и подсветкой XML синтаксиса. Программа считывает конфигурационный файл во
время запуска, таким образом вносить изменения в файл нужно перед ее стартом.

К программе может подключаться специальное оборудование: сканер штрих кодов,
фотокамера, весы, принтеры для печати этикеток и форм. После того, как вы 
определитесь с типом и набором используемого оборудования, отредактируйте 
раздел "Hardware" конфигурационного файла.

Для работы программы требуется установить подключение к сервису Aplix Post.
Настройки данного подключения указываются в разделе "PostService".

4. ЗАПУСК

Для запуска программы требуется запустить скрипт packline.bat или packline.sh
(в зависимости от типа операционной системы). Во время работы программы в 
консоль будет выводиться подробный лог и информация о возможных ошибках.

Под операционной системой Windows можно запустить скрипт run.vbs, тогда 
программа запустится без консоли.
